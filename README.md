# Car Repair Reservation System

Bu proje, sanayiler için hazırladığım araç tamir rezervasyonu sistemimin giriş bölümünü içermektedir.

## Proje İçeriği

- AspNetCoreApi
- Identity
- Swagger
- RESTful API
- JsonWebToken
- Authorization & Authentication
- Login
- Register
- AspNetCore 8.0
- FluentValidation
- CleanArchitecture
- DDD (Domain Driven Design)
- DTOLayer
- AspNetCore MVC 8.0
- EntityFrameworkCore
- MailValidation
